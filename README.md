# 🚗 Car Market Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes a car dataset to uncover pricing trends, market distribution, and key factors influencing vehicle value.

The workflow includes:
- Data cleaning
- Feature engineering
- Exploratory analysis
- Dashboard creation in Excel

---

## 🧹 Data Cleaning
Performed the following steps:
- Removed duplicates
- Handled missing values (e.g., engine volume)
- Removed outliers (price, mileage, engine volume)
- Standardized categorical variables (engine type, body type)
- Converted Year → Car Age

---

## ⚙️ Feature Engineering
Created new variables:
- Car Age
- Price Category (Low, Medium, High)
- Mileage Category
- Engine Size Category

---

## 📊 Dashboard
An interactive Excel dashboard was built using:
- Pivot Tables
- Charts
- Slicers

### Key Metrics:
- Average Price
- Total Listings
- Average Mileage
- Market distribution by body type and engine type

---

## 🔍 Key Insights
- Cars depreciate fastest within the first 5 years
- SUVs dominate the market
- Lower mileage vehicles command higher prices
- Engine type influences resale value
